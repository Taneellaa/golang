# Todo API

Простое REST API для управления задачами.

## Установка

1. Установите Go версии 1.16 или выше.
2. Клонируйте репозиторий: `git clone https://github.com/Taneellaa/tasks-crud.git`
3. Перейдите в каталог проекта: `cd tasks-crud`
4. Установите зависимости: `go mod download`

## Запуск

1. Создайте файл `.env` в корневом каталоге проекта и добавьте следующие переменные среды:
   - `PORT` - порт, на котором будет запущен сервер (например, `8080`)
2. Запустите сервер: `go run cmd/api/main.go`

## Использование

API предоставляет следующие эндпоинты:

- `GET /tasks` - получить список всех задач
- `GET /tasks/{id}` - получить задачу по идентификатору
- `POST /tasks` - создать новую задачу
- `PUT /tasks/{id}` - обновить существующую задачу
- `DELETE /tasks/{id}` - удалить задачу по идентификатору

## Документация

Документация API доступна по следующим URL-адресам:

- [Swagger UI](http://localhost:8080/swagger/index.html)
- [Swagger JSON](http://localhost:8080/swagger/doc.json)

## Тестирование

Для тестирования API можно использовать следующие инструменты:

- Postman
- cURL
